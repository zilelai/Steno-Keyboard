To make your own Steno Keyboard, follow these steps:

1. Components
   -24 12mm/6mm Buttons
   -Breadboard and Jumper wires
   -LCD I2C
   -Arduino Mega 2560

2. Wiring
   -Wire all the buttons base on the code
   -Wire SCL on the LCD I2C to A5/21 and SDA to A4/20

3. Coding
   -Use the code provided

4. Enjoy :)
